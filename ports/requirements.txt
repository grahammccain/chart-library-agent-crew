# repo root needs. Install both:

# port also CONSTRUCTS its framework objects - so you can verify a port for free. A
# --live run additionally needs that framework's model key (Anthropic / OpenAI).
#
# Each port hard-imports only its own framework; install just the one(s) you want to try.
